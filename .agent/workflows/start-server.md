# Iniciar Servidor Local

**IMPORTANTE**: Nunca tentar executar automaticamente. Sempre fornecer o comando para o usuário colar no PowerShell.

## Comando para o usuário:

```powershell
cd c:\ProAG\puretego-crm
python run.py
```

## URLs de acesso:
- **Local**: http://localhost:5000
- **Rede**: http://192.168.100.43:5000

## Para parar o servidor:
Pressione `Ctrl+C` no terminal.
